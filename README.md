Aqui irá aparecer a descrição do projeto e as RNs.




## Instalação

Para iniciar o projeto é necessário rodar o comando abaixo na raiz do projeto.

```bash
  docker-compose up -d
```

Em seguida, a aplicação estará disponível nos seguintes endereços

Back-end
```bash
  http://localhost:8080
```

Front-end
```bash
  http://localhost:3000
```
